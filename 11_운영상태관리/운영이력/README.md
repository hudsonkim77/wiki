# 운영이력 (OPS_CI_MAP)

## 개요
이 폴더는 운영상태관리(`OPS_STATUS`)와 구성관리(`CI`) 사이의 **N:M 관계를 연결하는 중간(연결) 테이블**을 보관합니다.
상위 도메인 설명은 [11_운영상태관리](../README.md), 구성관리 설명은 [3_구성관리](../../3_구성관리/README.md)를 참고하세요.

## 파일 목록
| 파일명 | 설명 | 건수 |
|---|---|---|
| OPS_CI_MAP.csv | 어떤 점검이 어떤 CI(들)를 대상으로 했는지 매핑 | 0건(스키마만 우선 반영) |

## 필드 정의
| 필드명 | 설명 | 비고 |
|---|---|---|
| MAP_ID | 매핑 고유 ID | **PK** |
| OPS_ID | 운영상태 점검 ID | **FK → ../OPS_STATUS.csv** |
| CI_ID | 점검 대상 구성항목 ID | **FK → ../../3_구성관리/CI.csv** |
| NOTE | 연결 근거 메모 | |

## 참고 및 관리 방법
- 점검 실행 시 [`../OPS_STATUS.csv`](../OPS_STATUS.csv)에 마스터 행을 먼저 추가하고, 점검 대상 CI를 이 표에 (OPS_ID, CI_ID) 쌍으로 행 단위 추가합니다.
