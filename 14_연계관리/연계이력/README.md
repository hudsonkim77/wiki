# 연계이력 (INTERFACE_CI_MAP)

## 개요
이 폴더는 연계관리(`INTERFACE`)와 구성관리(`CI`) 사이의 **N:M 관계를 연결하는 중간(연결) 테이블**을 보관합니다.
상위 도메인 설명은 [14_연계관리](../README.md), 구성관리 설명은 [3_구성관리](../../3_구성관리/README.md)를 참고하세요.

## 파일 목록
| 파일명 | 설명 | 건수 |
|---|---|---|
| INTERFACE_CI_MAP.csv | 어떤 연계가 어떤 내부 CI를 통하는지 매핑 | 0건(스키마만 우선 반영) |

## 필드 정의
| 필드명 | 설명 | 비고 |
|---|---|---|
| MAP_ID | 매핑 고유 ID | **PK** |
| INTERFACE_ID | 연계 ID | **FK → ../INTERFACE.csv** |
| CI_ID | 관련 내부 구성항목 ID | **FK → ../../3_구성관리/CI.csv** |
| NOTE | 연결 근거 메모 | |

## 참고 및 관리 방법
- 연계 등록 시 [`../INTERFACE.csv`](../INTERFACE.csv)에 마스터 행을 먼저 추가하고, 관련 내부 CI를 이 표에 (INTERFACE_ID, CI_ID) 쌍으로 행 단위 추가합니다.
