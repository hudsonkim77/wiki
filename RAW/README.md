## 참고 및 관리 방법
- 원본 데이터는 직접 수정하지 않고, 가공이 필요한 경우 별도 사본(백업)을 만들어 작업합니다.
- 새 원본 파일을 추가하거나 기존 파일을 정제할 때는 위 "파일 목록"과 "작업 업데이트 이력" 표를 함께 갱신합니다.
