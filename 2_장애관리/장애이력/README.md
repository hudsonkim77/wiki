# 장애이력 (INCIDENT_CI_MAP)

## 개요
이 폴더는 장애관리(`INCIDENT`)와 구성관리(`CI`) 사이의 **N:M 관계를 연결하는 중간(연결) 테이블**을 보관합니다.
설계 근거와 ERD는 [3_구성관리](../../3_구성관리/README.md), 상위 도메인 설명은 [2_장애관리](../README.md)를 참고하세요.

## 파일 목록
| 파일명 | 설명 | 건수 |
|---|---|---|
| INCIDENT_CI_MAP.csv | 어떤 CI가 어떤 장애에 연결되어 있는지 매핑 | 1건 — `MAP_0001`(`INC_20260711_001` ↔ `CFG_WEB_019`) |

## 필드 정의
| 필드명 | 설명 | 비고 |
|---|---|---|
| MAP_ID | 매핑 고유 ID | **PK** |
| INCIDENT_ID | 장애 ID | **FK → ../INCIDENT.csv** |
| CI_ID | 영향받은 구성항목 ID | **FK → ../../3_구성관리/CI.csv** |
| IMPACT_DESC | 해당 CI 기준 영향 내용 | |
