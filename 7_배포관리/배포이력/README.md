# 배포이력 (DEPLOY_CHG_MAP)

## 개요
이 폴더는 배포관리(`DEPLOY`)와 변경관리(`CHANGE`) 사이의 **N:M 관계를 연결하는 중간(연결) 테이블**을 보관합니다.
상위 도메인 설명은 [7_배포관리](../README.md), 변경 도메인 설명은 [1_변경관리](../../1_변경관리/README.md)를 참고하세요.

## 파일 목록
| 파일명 | 설명 | 건수 |
|---|---|---|
| DEPLOY_CHG_MAP.csv | 어떤 변경 티켓이 어떤 배포에 포함됐는지 매핑 | 0건(스키마만 우선 반영) |

## 필드 정의
| 필드명 | 설명 | 비고 |
|---|---|---|
| MAP_ID | 매핑 고유 ID | **PK** |
| DEPLOY_ID | 배포 ID | **FK → ../DEPLOY.csv** |
| CHG_TICKET_ID | 변경 티켓 ID | **FK → ../../1_변경관리/CHANGE.csv** |
| NOTE | 연결 근거 메모 | |

## 참고 및 관리 방법
- 배포 계획 확정 시 [`../DEPLOY.csv`](../DEPLOY.csv)에 마스터 행을 먼저 추가하고, 포함되는 변경 티켓을 이 표에 (배포 ID, 변경 티켓 ID) 쌍으로 행 단위 추가합니다.
