# 문제이력 (PROBLEM_INCIDENT_MAP)

## 개요
이 폴더는 문제관리(`PROBLEM`)와 장애관리(`INCIDENT`) 사이의 **N:M 관계를 연결하는 중간(연결) 테이블**을 보관합니다.
상위 도메인 설명은 [6_문제관리](../README.md), 장애 도메인 설명은 [2_장애관리](../../2_장애관리/README.md)를 참고하세요.

## 파일 목록
| 파일명 | 설명 | 건수 |
|---|---|---|
| PROBLEM_INCIDENT_MAP.csv | 어떤 장애가 어떤 문제로 이어졌는지(또는 문제 조사에 참고됐는지) 매핑 | 0건(스키마만 우선 반영) |

## 필드 정의
| 필드명 | 설명 | 비고 |
|---|---|---|
| MAP_ID | 매핑 고유 ID | **PK** |
| PROBLEM_ID | 문제 ID | **FK → ../PROBLEM.csv** |
| INCIDENT_ID | 장애 ID | **FK → ../../2_장애관리/INCIDENT.csv** |
| NOTE | 연결 근거 메모 | |

## 참고 및 관리 방법
- 문제 등록 시 [`../PROBLEM.csv`](../PROBLEM.csv)에 마스터 행을 먼저 추가하고, 관련 장애가 여러 건이면 이 표에 (문제 ID, 장애 ID) 쌍을 행 단위로 추가합니다.
